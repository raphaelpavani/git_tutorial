What is Git?

Git is a version control system that allows you to track changes in your code and collaborate with other developers. With Git, you can create a history of changes, experiment with new ideas, and revert to previous versions if something goes wrong.

Creating a Git Repository

Before you can start using Git, you need to create a Git repository. A Git repository is a folder where you keep your code and all the versions of that code. You can create a new Git repository by following these steps:

  1. Open your terminal or command prompt.
  2. Navigate to the folder where you want to create your Git repository.
  3. Type git init and press Enter. This will initialize a new Git repository in the current folder.

Adding Files to the Repository

Once you have created a Git repository, you can add files to it. In this example, we will add an HTML file and a CSS file to the repository. Here's how:

  1. Create a new file named index.html in your project folder.
  2. Create a new file named style.css in your project folder.
  3. Open your terminal or command prompt.
  4. Navigate to your project folder.
  5. Type git add index.html and press Enter. This tells Git to add the index.html file to the repository.
  6. Type git add style.css and press Enter. This tells Git to add the style.css file to the repository.

Committing Changes

After adding files to the repository, you need to commit the changes to create a snapshot of the repository at that point in time. Here's how:

  1. Open your terminal or command prompt.
  2. Navigate to your project folder.
  3. Type git commit -m "Initial commit" and press Enter. This creates a new commit with the message "Initial commit".

Working with Branches

A branch is a copy of your repository that allows you to work on a new feature or experiment with new ideas without affecting the main branch. Here's how to create and switch to a new branch:

  1. Open your terminal or command prompt.
  2. Navigate to your project folder.
  3. Type git branch feature-branch and press Enter. This creates a new branch named "feature-branch".
  4. Type git checkout feature-branch and press Enter. This switches to the "feature-branch" branch.

Merging Branches

After you have made changes to a branch, you can merge those changes back into the main branch. Here's how:

  1. Open your terminal or command prompt.
  2. Navigate to your project folder.
  3. Type git checkout main and press Enter. This switches to the main branch.
  4. Type git merge feature-branch and press Enter. This merges the changes from the "feature-branch" branch into the main branch.

Pushing Changes to GitHub

GitHub is a popular platform for hosting Git repositories and collaborating with other developers. Here's how to push your changes to a GitHub repository:

  1. Create a new repository on GitHub.
  2. Copy the repository URL.
  3. Open your terminal or command prompt.
  4. Navigate to your project folder.
  5. Type git remote add origin and press Enter. This tells Git to add the GitHub repository as a remote location.
  6. Type git push -u origin main and press Enter. This pushes the changes in the local main branch to the GitHub repository.
